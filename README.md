生命游戏（Python 编写）
-------------------------------------

用 Pygame 编写的生命游戏，可以改变大小并能产生随机游戏。

操作：

+ b键：开始游戏
+ s键：暂停游戏
+ c键：改变大小（在暂停游戏的情况下，输入格式：列数（每一行多少个细胞）,行数（每一列多少个细胞），回车键确定）
